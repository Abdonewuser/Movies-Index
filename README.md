# 🎬 Movie Index — React Movie Search Application

A movie discovery application built using React that allows users to browse and search for movies using a public movie database API.

The application fetches movie data dynamically and displays detailed information for each movie through a structured multi-page interface.

## 🚀 Features

- Browse popular movies
- Search movies by name
- View detailed movie information
- Dynamic routing for movie pages
- API data fetching and rendering
- Responsive UI layout

## 🛠 Tech Stack

- React (Hooks)
- React Router DOM
- JavaScript (ES6+)
- Movie Database API
- CSS3

## 🎯 What I Learned

- Fetching and handling API data in React
- Implementing dynamic routing for individual movie pages
- Managing application state for search functionality
- Rendering lists of API data efficiently
- Structuring a multi-page React application
